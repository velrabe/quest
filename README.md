# Quest — лаборатория на удалённой базе

Unity-проект и интерактивные HTML-прототипы для планирования локаций и геймплея.

## Прототипы (GitHub Pages)

- [Игра — 2D прототип](https://velrabe.github.io/quest/)
- [Карта территории](https://velrabe.github.io/quest/world-map.html)

Исходники прототипов: папка `Design/`.

## Unity

Откройте проект в Unity 6 (URP). Сцены и ассеты — в `Assets/`.
